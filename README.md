# tictactoe-eth

> Play tic tac toe - with Solidity

## scripts

### Install deps

```sh
npm install
cd frontend
npm install
cd ..
```

### Checks

- Test contracts: `npm test`
- Lint: `npm run check`

### Deploy contract

#### Local deploy

1. Start Hardhat's testing network: `npm run hardhat:node`
2. Deploy contract locally; while testing network is running: `npm run deploy:contract:localhost`

### Start frontend

```sh
cd frontend
npm start
```
